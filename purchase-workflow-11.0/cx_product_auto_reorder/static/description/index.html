How it works

Templates are searched and applied in the following order:

  1. Local Attributed*. These type of templates have BOTH Product Category AND Product Attribute Values defined.
    Example: When product "Laptop (Color:White,Memory:8Gb)" with category "All/Saleable/Laptops" is created Odoo will search for template with BOTH Category="All/Saleable/Laptops" AND Attribute Values=[Color:White AND Memory 8Gb] and apply it.
  2. Local. Local templates have ONLY Product Category defined.
    Example: In case no Local Attributed* template for our Laptop was found Odoo will search for template with Category="All/Saleable/Laptops" AND NO Attribute Values defined and apply it.
  3. Global Attributed*. These type of templates have ONLY Product Attribute Values defined.
    Example: In case no Local template for our Laptop was found Odoo will search for template with NO Category defined AND Attribute Values=[Color:White AND Memory 8Gb] and apply it.
  4. Global. This template has NO Product Category AND NO Product Attribute Values defined. Use this template as a fallback in case no other template can be applied.
* Attributed templates are available in PRO version

Set up Reordering Rules Templates

Go to "Inventory -> Master Data -> Reordering Rule Templates

Create new Rule Template

1. Select Product Category and/or Product Attribute Values* (* Attribute Values are available in PRO version).
2. Configure rule settings (min/max qty, warehouse, company etc.).
3. Do not forget: Category and Product Attribute Values can be specified ONLY when Template is created!
4. Click "Save".

Rules are created automatically

Note: "Apply rules" is used only if you modified some Rules manually and want to reset them later

Modify Template to update Rules

Rules are updated automatically when Template is changed

Rules are re-assigned when Template is deleted

1. Rules are automatically re-assigned to the other matching Template when Template is deleted.
2. You can manually assign Reordering Rule to any Template.
3. Or you can uncheck "Control via Template" and disable Rule modification using Template.

Create new Product

1. Select Product Category and/or Product Attribute Values* (* Attribute Values work in PRO version).
2. Check "Don't create reordering rules from templates" if you do not want to create Reordering Rules automatically.
3. Click "Save".

Rules are created automatically

Rules are changed when Product is updated

New Templates is applied and Reordering Rules are updated automatically when Product Category and/or Product Attribute Values* are changed (* Attribute Values are available in PRO version)
