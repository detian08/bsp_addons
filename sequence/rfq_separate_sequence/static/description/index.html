User Guide

Once you install the module a new sequence under Technical Sequences will be added with the name RFQ Purchase, this will be used as a separate sequence for the newly created RFQ.

1 & 2 are the 'Prefix' and 'Suffix' which will be added to the beginning and ending of the RFQ number if filled.
3 is the length of the sequence after the 'Prefix' and before the 'Suffix'.
4 Indicates how much the next sequence will increase.


After confirmation, it turns into a separate PO and also upon cancellation and reset to draft, it will take the original RFQ sequence.
